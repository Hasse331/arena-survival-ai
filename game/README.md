# Arena Survival Game

Simple web-based arena survival game built with Phaser 3, TypeScript, and Vite.

## Features

- Start screen with two buttons
- `Play as Human` starts the game
- `Play with Bot` is visible but intentionally inactive
- Real-time movement with `WASD` or arrow keys
- Endless enemy spawning with increasing pressure
- Health, score, survival timer, and game over screen

## Run

```bash
npm install
npm run dev
```

Open the local Vite URL in your browser to play.

## Controls

- `WASD` or arrow keys: move
- `Esc`: return to menu
- `Space` on game over: restart
